# Backend Setup Guide

This document explains how to set up and run the backend server of the project.

---

## Prerequisites

- Node.js (v14 or higher recommended)

- MongoDB Community Server  
  Make sure MongoDB is running on port 27017 (default).

- MongoDB Compass (optional)  
  GUI tool for managing MongoDB databases.

---

## Environment Variables Setup

1. Create a `.env` file in the backend folder (copy from `.env.example`):

   ```bash
   cp .env.example .env
   ```
